# Teamspeak3MusicBot

## Syntax

| Function                     | Command                      |              
| ---                          | ---                          | 
| playing a song from youtube  | "!play example.url"          |
| playing the same song again  | "!play"                      |
| pausing the song             | "!pause"                     |
| stopping the song            | "!stop"                      |
